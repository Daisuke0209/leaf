# Google Workspace Marketplace 掲載素材(コピペ用)

## App name
Leaf

## Short description (最大 ~200字)
A Markdown editor for Google Drive with a Notion-style writing experience.
Your notes stay as plain .md files in your own Drive — readable anywhere,
with real-time collaboration built in.

## Detailed description
Leaf turns Google Drive into a beautiful Markdown workspace.

✍️ Notion-style editing — blocks, headings, lists, checkboxes and more,
with a clean distraction-free design.

📄 Plain Markdown files — every page is a normal .md file in your Drive.
No lock-in: open them in any editor, feed them to NotebookLM, grep them,
version them. Your data never leaves your Drive.

🤝 Real-time collaboration — live cursors, presence avatars and instant
sync. Access follows your Drive sharing: editors can write, viewers get
read-only, and revoking access in Drive revokes it in Leaf.

🔒 Privacy by design — Leaf has no database and no analytics. It only
sees files you create with it or open with it (drive.file scope), and
runs entirely in your browser.

Getting started:
1. In Drive, choose New → More → Leaf to create a page, or
2. Right-click any Markdown file → Open with → Leaf.
